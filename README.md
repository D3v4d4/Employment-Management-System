The Employment Management System (EMS) is a web-based application designed to automate and manage essential Human Resource (HR) and payroll operations in an organization. The system centralizes employee data, simplifies attendance and leave management, automates payroll calculations, and enables smooth internal communication through an integrated chat module.
This project was developed as part of the BCA (Bachelor of Computer Applications) program at Indira Gandhi National Open University (IGNOU) for academic purposes.
